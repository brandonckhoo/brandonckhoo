## 💡 How I Ship

1. **Define the bet** — 1-sentence hypothesis: user, moment, expected behavior, primary metric, 2 guardrails, decision date
2. **Map workflow failures** — Happy path + top 3 failure paths; find trust breakpoints
3. **Prototype in code** — Claude Code for scaffold; Cursor for UX; Vercel preview in 24–72h
4. **Validate with customers** — 5–10 tests; pair calls with session replay
5. **Ship thin slice** — Smallest version delivering 1 clear outcome; feature flags
6. **Iterate to stable** — Segment by cohort, remove friction, invest in scale levers
